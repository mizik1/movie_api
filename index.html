Welcome to My Flix API!

REQUEST ENDPOINT URL METHOD TYPE FORMAT OF REQUEST DATA FORMAT OF RESPONSE DATA
Return a list of ALL movies to the user. app.get("/movies") GET - READ None A JSON object holding data for all movies.
Return data about a single movie by title. app.get("/movies/title") GET - READ None A JSON object holding data about the requested movie.
Example: "title":Scarface
Endpoint: movies/title/Scarface
Return data about a movie genre. app.get("movies/genre") GET - READ None A JSON object holding data about the requested genre.
Example: "genre":Comedy
Endpoint: movies/genre/Comedy
Return data about a director. app.get("movies/director") GET - READ None A JSON object holding data about the requested director.
Example: "director":John Carpenter
Endpoint: movies/director/John Carpenter
Let user add new movie. app.post("/users/favorites/movieId") POST None A JSON object to add user's favorite movie.
Example: "id": "Shrek"
Endpoint: ("/users/favorites/Shrek")
Let user delete a movie they previously entered. app.delete("/users/favorites/movieId") DELETE None A JSON object to remove movie that was previously entered.
Example: "id": "Shrek"
Endpoint: ("/users/favorites/Shrek")
Let new user create user ID. app.put("/users/id") PUT None A JSON object with new user data.
Example: "id": "MaryAdams, "email": "mary@gmail.com"
Endpoint: ("/users/MaryAdams")
Let user delete user ID. app.delete("/users/id") DELETE None A JSON object to delete user data.
Example: "id": "MaryAdams", "email": "mary@gmail.com"/
Endpoint: ("/users/MaryAdams")
Let user update their information. app.put("/users/id") PUT None A JSON object to update user's information.
Example: "id": "Mary Adams" changes last name to "Mary Jones"
Endpoint: ("/users/MaryJones")
